int16ToHex

src/creator/createCompactLog.js

A log entry is a hexadecimal line composed of :

  • a sequential ID (8)
  • epoch (8)
  • a list of parameters values (n * 4)
  • a log event ID (4)
  • a log event value (4)
  • a device ID (4)
  • a checkdigit (2)

This means that for 26 parameters, the length of a log is 134 hexadecimal characters.
